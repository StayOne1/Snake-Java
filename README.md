# Snake-Java


A simple snake game

这款游戏的开发是我目前写的第一个java游戏项目，是我跟着麦叔的视频做的，在此也要说一下，麦叔的这个教程是比较清晰的，将这个游戏的各个部分讲解的比较详细

游戏已经打包为jar文件，在java环境下可以直接执行
